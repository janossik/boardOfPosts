### Dokumentacja api

**_Typ: get_**

endpoint:

```
/
```

### Pobierz wszystkie posty

**_Typ: get_**

endpoint:

```
/posts
```

### Pobierz jeden post

**_Typ: get_**

endpoint:

```
/posts/:id
```

### Dodaj post

**_Typ: POST_**

endpoint:

```
/posts/add
```

### Edytuj post

**_Typ: PUT_**

endpoint:

```
/posts/edit
```

### Usuń post

**_Typ: DELETE_**

endpoint:

```
/posts/remove
```
